# decodelabs-task-2

this is my secod project of decodelabs. in this project i have created responsive static webpage and this project represents the vital bridge between a static site and a professional, multi-device web experience.


​
​*)Project Goal

​The objective of this project was to master the art of Fluidity. Moving beyond fixed dimensions, I built a webpage that gracefully transitions across mobile, tablet, and desktop screens using pure CSS logic.

​*) Key Features & Skills

​CSS Media Queries: Implemented precise breakpoints to ensure layout integrity on all devices.
​Flexbox & Grid: Used modern layout techniques for professional spacing and alignment.
​Responsive Navigation: Designed a menu that adapts to screen real estate.
​Fluid Grids: Focused on relative units (em, rem, %) instead of fixed pixels to achieve a seamless user experience.


​*)Design Philosophy

​"Build from the content out." This project represents the vital bridge between a static site and a professional, multi-device web experience.
